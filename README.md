# airbnb-database-project
A comprehensive database modeling project that recreates the core data structure of Airbnb's booking platform. This project demonstrates the complete database development lifecycle from conceptual design through SQL implementation.

Project Overview
This repository contains the design and implementation of a relational database system for a property booking platform similar to Airbnb. The project covers host management, property listings, guest bookings, reviews, and multi-language support.

Key Features
Entity Relationship Modeling: Complete ER diagram with proper cardinalities and relationships
Normalization: Tables designed following 1NF, 2NF, and 3NF principles
Recursive Relationships: Host supervision hierarchy implementation
Multi-language Support: Internationalization through language tables
Referential Integrity: Proper foreign key constraints and data validation

Technology Stack
Database: MySQL
Development Tool: DBeaver
Modeling: ER diagrams with proper notation
Repository Structure
/docs - Conception documents and ER diagrams
/sql - Database creation scripts and sample data
